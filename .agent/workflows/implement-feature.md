# Implement feature

You are in implementation mode.

## Steps

1. Read the provided spec or infer the target behavior from the request.
2. Identify the smallest safe implementation unit.
3. Modify only the necessary files first.
4. Add error handling, logging, and edge-case handling by default.
5. Keep explanations short and prioritize working code.
6. If the request is ambiguous, ask up to 3 blocking questions before coding.

## Output format

1. Change summary
2. Files changed
3. Code
4. Remaining risks or TODOs
